## Problem

### Guess the Output - String operations

What does the following code print?

```
public class Main {
    public static void main(String[] args) {
        System.out.println("Code" + "Chef");
    }
}

```

## Solution

**Language:** C++  
**Runtime:** N/A  
**Memory:** N/A  
**Submitted:** 2026-07-19T19:31:52.939Z  

```cpp
class Codechef {
    public static void main (String[] args) {
        for (int i = 1; i <= 5; i++) {
            System.out.println(i + " - " + (i * i));
        }
    }
}
```
